Overview

The MapleStory Account Tracker helps you monitor character progression across multiple aspects of the game:

  • Character levels and basic information
  • Arcane Symbol progression
  • Sacred Symbol progression
  • Equipment status
  • Cash shop items

Navigation

Use the navigation bar at the top to access different views:

  • Overview: Basic character information and symbol totals
  • Progression: Detailed character information including faction and class
  • Equipment: Equipment status for each character
  • Cash: Cash shop item tracking
  • Arcane Symbol Detail: Detailed view of Arcane Symbol levels
  • Sacred Symbol Detail: Detailed view of Sacred Symbol levels

Data Files

The tracker uses CSV files in the data/ folder:

  • account.csv: Basic character information
  • joblist.csv: Job/class information
  • arcane.csv: Arcane Symbol levels
  • sacred.csv: Sacred Symbol levels
  • equipment.csv: Equipment tracking
  • cash.csv: Cash shop items

Updating Data

To update character information:

  1. Open the relevant CSV file in the data/ folder
  2. Add or modify entries as needed
  3. Save the file and refresh the page to see updates
