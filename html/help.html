Overview

The MapleStory Account Tracker helps you monitor character progression across multiple aspects of the game:

  • Character levels and basic information
  • Arcane Symbol progression
  • Sacred Symbol progression
  • Equipment status
  • Cash shop items

Navigation

Use the navigation bar at the top to access different views:

  • Overview: Basic character information and symbol totals
  • Help: Documentation and usage guides
  • Progression: Detailed character information including faction and class
  • Equipment ‚ñº: Dropdown menu for equipment management
    • Armor: Core equipment pieces (weapons, armor, etc.)
    • Accessory: Additional equipment (rings, pendants, etc.)
    • Cash Shop: Cash shop items and NX equipment
  • Symbols ‚ñº: Dropdown menu for symbol progression
    • Arcane: Arcane Symbol levels
    • Sacred: Sacred Symbol levels

Data Files

The tracker uses CSV files in the data/ folder:

  • account.csv: Basic character information
  • joblist.csv: Job/class information
  • arcane.csv: Arcane Symbol levels
  • sacred.csv: Sacred Symbol levels
  • equipment.csv: Armor and main equipment tracking
  • accessory.csv: Accessory equipment tracking
  • cash.csv: Cash shop items

Equipment Organization

Equipment is organized into two categories for better tracking:

  • Armor: Main equipment pieces including:
    • Weapons and Secondary weapons
    • Emblem
    • Main armor pieces (Hat, Top, Bottom, Shoes, Cape, Gloves, Shoulder)
  • Accessories: All additional equipment slots including:
    • Face, Eye, and Ear accessories
    • Rings (4 slots)
    • Pendants (2 slots)
    • Belt
    • Badge and Medal
    • Android and Heart

Special equipment types are color-coded in the interface:

  • Superior items (gold)
  • Meister items (silver)
  • Reinforced items (bronze)
  • AbsoLab/Absolab items (grey)
  • Arcane/Umbra items (purple)

Updating Data

To update character information:

  1. Open the relevant CSV file in the data/ folder
  2. Add or modify entries as needed
  3. Save the file and refresh the page to see updates

For equipment updates:

  • Use equipment.csv for core equipment pieces (weapons, armor)
  • Use accessory.csv for accessories and additional equipment
  • Make sure to match the equipment names exactly for proper color coding
  • Leave cells empty if the slot is not filled rather than using "None" or "0"
