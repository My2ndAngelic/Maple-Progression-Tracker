Overview

The MapleStory Account Tracker helps you monitor character progression across multiple aspects of the game:

  • Character levels and basic information
  • Job class and archetypes
  • Arcane Symbol progression and bonus stats
  • Sacred Symbol progression and bonus stats
  • Equipment status across armor, accessories and cash items
  • Inner Ability lines across all character presets

Navigation

Use the navigation bar at the top to access different views:

  • Overview: Basic character information and symbol totals
  • Progression: Detailed character information including faction and class
  • Equipment ‚ñº: Dropdown menu for equipment management
  • Inner Ability: Character inner ability lines for all presets
    • Armor: Core equipment pieces (weapons, armor, etc.)
    • Accessory: Additional equipment (rings, pendants, etc.)
    • Cash: Cash shop items and NX equipment
  • Symbols ‚ñº: Dropdown menu for symbol progression
    • Arcane: Arcane Symbol levels
    • Sacred: Sacred Symbol levels
  • Help: Documentation and usage guides (this page)

Data Files

The tracker uses CSV files in the data/ folder:

  • account.csv: Basic character information
  • joblist.csv: Job/class information
  • arcane.csv: Arcane Symbol levels
  • sacred.csv: Sacred Symbol levels
  • equipment.csv: Armor and main equipment tracking
  • accessory.csv: Accessory equipment tracking
  • cash.csv: Cash shop items

Class Organization

Characters are organized by their archetypes in the Progression view, with color coding for easy identification:

  • Warrior (Red)
  • Magician (Blue)
  • Archer (Green)
  • Thief (Purple)
  • Pirate (Light Brown)

Equipment Organization

Equipment is organized into three categories for better tracking:

  • Armor: Main equipment pieces including:
    • Weapons and Secondary weapons
    • Emblem
    • Main armor pieces (Hat, Top, Bottom, Shoes, Cape, Gloves, Shoulder)
  • Accessory: All additional equipment slots including:
    • Face, Eye, and Ear accessories
    • Rings (4 slots)
    • Pendants (2 slots)
    • Belt
    • Badge and Medal
    • Android and Heart
  • Cash: Cash shop items and cosmetics

Special equipment types are color-coded in the interface:

  • Superior items (gold)
  • Meister items (silver)
  • Reinforced items (bronze)
  • AbsoLab/Absolab items (grey)
  • Arcane/Umbra items (purple)
  • Princess No Secondary (special highlight)

Keyboard Navigation

The dropdown menus can be navigated using:

  • Tab key to move between navigation elements
  • Arrow keys to navigate dropdown options
  • Enter or Space to select an option
  • Escape to close a dropdown without selecting

Inner Ability

The Inner Ability page shows you all character inner abilities across three presets:

  • Each character has up to 3 presets (P1, P2, P3)
  • Each preset has 3 ability lines (Line 1, Line 2, Line 3)
  • Inner abilities are color-coded by type for easier reference:
    • Attack Speed (AS+): Light red
    • Boss Damage (Boss+/BD+): Light green
    • Buff Duration (Buff+): Light blue
    • Cooldown Skip (CDSkip+): Light purple
    • Meso Obtain (Meso+): Light yellow
    • Item Drop (Item+): Light cyan
    • Passive Skill (Passive+): Light violet
    • Abnormal Status (Abnormal+): Light orange

Updating Data

To update character information:

  1. Open the relevant CSV file in the data/ folder
  2. Add or modify entries as needed
  3. Save the file and refresh the page to see updates

For equipment updates:

  • Use equipment.csv for armor and main equipment pieces
  • Use accessory.csv for accessory equipment
  • Use cash.csv for cash shop items
  • Use innerability.csv for character inner abilities
  • Make sure to match the equipment names exactly for proper color coding
  • Leave cells empty if the slot is not filled rather than using "None" or "0"
